## 项目说明

此项目用来维护整理机器学习的案例，包括传统的机器学习方法、深度学习、CNN、RNN、强化学习等内容。

## 第三方依赖

* Python 3.5
* Jupyter Notebook
* Numpy
* Matplotlib
* keras
* TensorFlow
* Pytorch

## 数据依赖

机器学习离不开数据，脱离数据的机器学习都是空谈。一般情况下的机器学习的教程，会针对一个机器学习框架展开。本项目针对每一个数据集，采用不同的机器学习方法和框架进行“学习”。为便于数据下载，会将数据统一保存到网盘。这样的做法好像是很 low，但简单粗暴，一是不会出现被“墙”的情况，二是可以做到数据的复用性。
